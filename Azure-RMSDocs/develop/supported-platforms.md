# <a name="supported-platforms"></a>受支持的平台

本主题介绍支持 Rights Management Services SDK 2.1 的客户端和服务器平台。

## <a name="supported-platforms"></a>受支持的平台

-   Windows 10
-   Windows Server 2012 R2
-   Windows Server 2012
-   Windows 8
-   Windows Server 2008 R2 SP1
-   Windows 7 + SP1

**注意**  从 Windows Server 2008 开始，支持模板功能的使用。
